# Vim
 
---

## Mode 

i -> insert mode
esc -> default mode

<br> 

---

## Save/Quit

:w -> save
:wq -> save and quit
:q! -> quit without save

<br>

---

## Navigation

gg -> go to the top
G -> go to the end
/ -> search pattern
:set number -> print number lines
:<num_line> -> go to number line

<br>

---

## Copy/Paste

yy -> copy line
:<start_line>,<end_line>y -> copy specific lines
p -> paste

<br>

---

## Delete

dd -> delete line

<br>

---

# Misc

:undo -> remove last operations
:redo -> replay last operations
