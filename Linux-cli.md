## Help

```bash
# Classical help
$ man <command>
$ <command> --help
$ <command> -h
```

```bash
# Specify man section
# 1. Shell's commands
# 2. Kernel API
# 3. Libraries
# 4. Special files
# 5. File format
# 6. Games, gadgets, etc.
# 7. Miscellaneous
# 8. Administrative commands
# 9. Sub program

# Example
# Get passwd file format
$ man 5 passwd 
```

```bash
# Retrieve command
$ man -k <pattern>
```
